# UTE-FV-0100 — Number Proof

## Claim

A number is an abstract representation of quantity, order, or relation.

## Plain-Language Explanation

A number allows quantity or order to be represented without requiring the original objects to be present.

For example, three stones and three marks are different things, but both may be represented by the same number.

## Dependencies

- UTE-FV-0002 — Equality of Quantity
- UTE-FV-0003 — Counting and Natural Numbers

## Reconstruction Method

A future intelligence can rediscover numbers by:
1. comparing collections,
2. counting distinguishable items,
3. preserving final count markers,
4. abstracting the marker away from the original objects.

## Conditions of Validity

The representation must preserve the relevant quantity, order, or relation.

## Failure Cases

A number is misused if its reference system is unclear or if quantity, order, and measurement are conflated.

## UTE Importance

Number supports arithmetic, algebra, measurement, computation, and scientific reconstruction.
