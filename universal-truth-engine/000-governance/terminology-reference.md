# UTE Terminology Reference

## Preferred Terms

- Core Truth
- Core Vault
- Reference Schema
- Truth Topology
- Dependency Link

## Deprecated Terms

- Canonical Truth
- Canonical Entry
- Canonical Structure

## Reason

The term 'canonical' may imply doctrinal authority or fixed orthodoxy.

The Universal Truth Engine prioritizes:
- evidence,
- revisability,
- transparency,
- and structured uncertainty.

The term 'core' better reflects foundational importance without implying infallibility.
