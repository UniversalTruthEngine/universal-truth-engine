# Arithmetic Substrate v1

## Purpose

Defines the first controlled M2-A arithmetic expansion layer.

## Truths Added

- UTE-FV-0100 — Number
- UTE-FV-0101 — Natural Number
- UTE-FV-0102 — Addition
- UTE-FV-0103 — Subtraction

## Dependency Direction

```text
Equality of Quantity

        ↓
Number
        ↓
Natural Number
        ↓

        ↓

```

## Governance Note
