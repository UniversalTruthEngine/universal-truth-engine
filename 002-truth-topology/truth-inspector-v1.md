# Truth Inspector v1

## Purpose

The Truth Inspector is the human and machine interface between the visual map and the underlying truth object.

Each node should allow inspection of:

1. Summary
2. Claim
3. Proof or derivation
4. Validation assessment
5. Reconstruction basis
6. Dependencies
7. Dependents
8. Stable coordinates
9. Machine-readable truth data

## Principle

A map node should not contain the full proof. It should point to inspectable proof layers.

The map is the navigational layer. The truth object is the epistemic object.
