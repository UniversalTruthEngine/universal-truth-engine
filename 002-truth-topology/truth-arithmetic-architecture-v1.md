# Truth Arithmetic Architecture v1

## Purpose

Describe the first UTE architecture for deriving possible next truths from known truth chains.

## Flow

```text
Truth Graph
    ↓
Truth Arithmetic Rules
    ↓
Derivation Chain Detection
    ↓
Candidate Next Truths
    ↓
Human + LLM Review
    ↓
Fact Vault Admission
```

## Example

```text
Addition
    ↓
Repeated addition
    ↓
Candidate: Multiplication
```

## Safety

The engine does not decide truth.

The engine identifies possible next truths for review.
