# Frontier Candidate Reporting Protocol v1

## Purpose

Make UTE frontier candidates visible to reviewers.

## Reporting Flow

```text
Automated Frontier Detection
    ↓
truth-frontier-report-v1.json
    ↓
Public docs/data copy
    ↓
frontier.html
    ↓
Human + LLM review
```

## Governance Rule

A displayed frontier candidate is not an accepted truth.

It is a review candidate only.
