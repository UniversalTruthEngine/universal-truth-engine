# Automated Frontier Detection Protocol v1

## Purpose

Define how UTE may detect candidate next truths from existing truth topology.

## Core Principle

The UTE may identify frontier candidates.

Only review can admit them.

## Current Method

Version 1 uses controlled arithmetic derivation rules.

Example:

```text
Division + Ratio
    ↓
Candidate: Fraction
```

## Review Required

Each candidate must pass:

- human review
- LLM assisted review
- critical review
- adversarial review
- UTE Assurance
- UTE Soundness Review

## Safety Rule

A frontier candidate is not a truth.

It is a proposed next truth awaiting review.
