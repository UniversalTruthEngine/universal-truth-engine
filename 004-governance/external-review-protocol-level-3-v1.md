# External Review Protocol Level 3 v1

Purpose: Independent critical review of a Core Truth.

Review Questions:
1. Is the statement actually true?
2. Is it fundamental?
3. Can it be reduced further?
4. Are dependencies justified?
5. Can it be reconstructed from first principles?
6. Are there hidden assumptions?
7. Would a domain expert object?
8. Can it be stated more clearly?

Output:
- Accept
- Accept with revisions
- Reject
