# Truth Template Standard v1

## Purpose

Defines the standard structure for Core Truth entries within UTE.

Recommended structure:

```text
UTE-FV-XXXX/
├── entry.md
├── proof.md
├── metadata.json
└── diagrams/
```

Each truth should exist as:
- human-readable,
- machine-readable,
- inspectable,
- dependency-aware,
- and reconstructable.
