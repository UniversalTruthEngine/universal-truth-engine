# Topology Generation Standard v1

The Fact Vault is the source of truth. Browser-facing topology files should be generated from `001-fact-vault/**/metadata.json` wherever possible.

Generated files include:

- `docs/data/truth-index-v1.json`
- `docs/data/truth-map-v1.json`
- `docs/data/topology-health-preview.json`
- `003-machine-readable/truth-graph-generated-v1.json`
