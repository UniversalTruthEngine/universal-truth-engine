# Core Truth Standard v1

## Purpose

Defines the structural standard for Core Truth entries within the Universal Truth Engine.

## Core Components

Each Core Truth should contain:

- entry.md
- proof.md
- metadata.json

within:

001-fact-vault/UTE-FV-XXXX/

## Goals

The standard exists to preserve:
- consistency,
- reconstructability,
- dependency integrity,
- machine readability,
- and long-term topology coherence.
