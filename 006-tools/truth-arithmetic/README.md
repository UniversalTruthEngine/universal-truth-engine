# Truth Arithmetic Engine v1

## Purpose

The Truth Arithmetic Engine identifies derivation chains between existing truths and proposes next truths for review.

## Run

```bash
python 006-tools/truth-arithmetic/truth-arithmetic-engine-v1.py
```

## Input

```text
003-machine-readable/truth-graph.json
```

or:

```text
docs/data/truth-map-v1.json
```

## Outputs

```text
003-machine-readable/truth-arithmetic-report-v1.json
003-machine-readable/truth-arithmetic-report-v1.md
docs/data/truth-arithmetic-report-v1.json
```

## Principle

Truth arithmetic proposes.

Review admits.
