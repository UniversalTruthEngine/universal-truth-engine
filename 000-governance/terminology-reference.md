# UTE Terminology Reference

## Purpose

This document defines preferred terminology for the Universal Truth Engine.

Terminology matters because UTE is intended to be interpreted by humans, artificial intelligences, and potentially non-human intelligences across long timescales.

The language should avoid unnecessary institutional, religious, or dogmatic implications.

---

## Preferred Terms

| Preferred Term | Meaning |
|---|---|
| Core Truth | A truth entry with foundational importance, high dependency value, or structural centrality |
| Core Vault | The high-priority layer of validated, reconstructable truths |
| Reference Schema | The standard structure used to describe truth entries |
| Truth Topology | The map of relationships between truths |
| Dependency Link | A directed relationship showing that one truth depends on another |
| Reconstruction Protocol | A method for rediscovering or validating a truth with minimal assumptions |
| Confidence Level | A graded assessment of epistemic strength |
| Review History | Append-only record of changes, reassessments, or disputes |

---

## Deprecated Terms

| Deprecated Term | Replacement |
|---|---|
| Canonical Truth | Core Truth |
| Canonical Entry | Core Entry |
| Canonical Structure | Core Structure |
| Canonical Truth Schema | Reference Truth Schema |

---

## Rationale

The term "canonical" can imply religious authority, institutional orthodoxy, or fixed doctrine.

UTE prioritises:

- evidence,
- revisability,
- transparency,
- reconstructability,
- and structured uncertainty.

The term "core" better reflects foundational importance without implying infallibility.

---

## Important Distinction

A truth may be:

- **core** because many other truths depend on it,
- **stable** because evidence for it is strong,
- **useful** because it has high predictive or reconstructive value.

These are related but not identical.

Example:

| Truth | Core? | Stable? |
|---|---|
| Equality | Extremely core | Extremely stable |
| Newtonian mechanics | Moderately core | Stable within domain |
| Semiconductor lithography | Less core | Stable but highly dependent |

---

## Approved Usage

Use:

```text
Core Truth
Core Vault
Core Entry
Reference Schema
Truth Topology
```

Avoid:

```text
Canonical Truth
Canonical Entry
Final Truth
Absolute Truth
Doctrine
```
