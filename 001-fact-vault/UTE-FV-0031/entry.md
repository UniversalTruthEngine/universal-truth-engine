# UTE-FV-0031 — Position

## Statement

Position is the location of an entity relative to a defined reference framework.

## Domain

Geometry / Reference

## Summary

Position is not the coordinate system itself.

A coordinate system provides the framework.

Position is the location expressed within that framework.

Multiple coordinate systems may describe the same position.

## Boundary Notes

Position requires reference.

Without a reference framework, location cannot be specified meaningfully.

## Stabilisation Status

sound and foundational, with boundary clarifications added
