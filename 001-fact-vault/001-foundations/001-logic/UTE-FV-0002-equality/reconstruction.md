# Reconstruction — UTE-FV-0002

## Minimum Tools

- Distinguishable objects
- Ability to compare or pair objects
- Stable attention or memory

## Practical Rediscovery Method

1. Place objects from group A beside objects from group B.
2. Pair each object from A with one object from B.
3. If no unpaired objects remain in either group, the groups are equal in number.
4. Add or remove one object from a group.
5. Observe equality is broken.
6. Restore the missing object.
7. Observe equality returns.
