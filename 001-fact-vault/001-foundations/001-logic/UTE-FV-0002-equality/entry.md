# UTE-FV-0002 — Equality / Identity of Quantity

## Claim

If two quantities are equal, then each may be substituted for the other in any valid statement without changing the truth of that statement, provided the context and meaning are preserved.

```text
If A = B, then any valid statement about A is also valid about B, within the same context.
```

## Confidence Level

Level 5 — Mathematical / Logical Proof

## Core Axioms

- Reflexivity: `A = A`
- Symmetry: if `A = B`, then `B = A`
- Transitivity: if `A = B` and `B = C`, then `A = C`
- Substitutability: if `A = B`, then `A` may replace `B` in valid contexts

## Conditions of Validity

- The compared entities are clearly defined.
- The domain of comparison is specified.
- The meaning of sameness is fixed for that context.
- Measurement tolerance is accounted for when physical quantities are involved.

## Failure Cases

- Similarity mistaken for equality
- Approximation mistaken for exact equality
- Physical identity confused with equality of property
- Context changes between statements

## Preservation Priority

Critical root truth for logic, arithmetic, measurement, proof, and computation.
