# UTE-FV-0012 — Roots

## Category
Foundations → Arithmetic

## Claim
A root is a value which, when multiplied by itself a specified number of times, produces a given quantity.

## Confidence Level
Level 5 — Mathematical / Logical Proof

## Core Status
Core Truth

## Conditions of Validity
- Quantities or relations must be defined clearly.
- The domain of use must be specified.
- Symbolic use must preserve meaning across transformations.
- For physical applications, measurement tolerance must be considered.

## Failure Cases
- Misuse outside the defined domain.
- Confusing exact symbolic equality with approximate measurement.
- Applying the relation while changing context or units.
- Treating a model or representation as the physical object itself.
