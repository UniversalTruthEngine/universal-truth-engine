# UTE-FV-0104 — Zero

## Statement

Zero represents the absence of counted quantity or the neutral reference point for additive quantity.

## Domain

Arithmetic

## Dependencies

- UTE-FV-0002
- UTE-FV-0003
- UTE-FV-0100
- UTE-FV-0101

## Plain-Language Explanation

Zero allows arithmetic to represent absence, balance, or a neutral additive reference point.

## Status

Core Truth.
