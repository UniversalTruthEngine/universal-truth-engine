# Proof — Zero

## Claim

Zero represents the absence of counted quantity or the neutral reference point for additive quantity.

## Plain-Language Explanation

If countable quantity can be represented, then a context may also require representation of no counted units. Zero provides this stable representation of absence or additive neutrality.

## Dependencies

- UTE-FV-0002
- UTE-FV-0003
- UTE-FV-0100
- UTE-FV-0101

## Reconstruction Method

Begin with countable quantity. Recognise that a defined counting context can contain no counted items. Represent this absence as a stable value. In additive contexts, this value acts as the neutral reference because adding no quantity does not change the represented quantity.

## Conditions of Validity

Valid where a counting, quantity, or additive reference context is defined.

## Failure Cases

Zero should not be confused with undefined quantity, non-existence of a context, or non-applicability. It only applies where absence or neutrality can be represented within a defined system.
