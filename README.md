# Universal Truth Engine

The Universal Truth Engine (UTE) is a long-horizon knowledge preservation project intended to preserve durable, verifiable, reconstructable truths for humans, future artificial intelligence systems, and any other intelligence capable of reasoning from evidence.

The project prioritises:

1. Reproducibility
2. Re-derivability
3. Minimal assumptions
4. Long-term interpretability
5. Human and machine readability
6. Dependency-aware knowledge structure
7. Reconstruction after technological or institutional disruption

## Origin

The Universal Truth Engine concept, including the idea of a navigable cosmological map of fundamental knowledge, was originated by John David Stiles.

## Core Components

- `000-governance/` — charter, admission standards, review process
- `001-fact-vault/` — canonical truth entries
- `002-truth-topology/` — map and relationship layer
- `003-machine-readable/` — global JSON graph and schemas
