# Universal Truth Engine

The Universal Truth Engine (UTE) is a long-term knowledge architecture designed to preserve, structure, validate, and reconstruct foundational truths across time.

## Repository Structure
- `000-introduction/` — introductory overview
- `000-governance/` — early governance and terminology
- `001-fact-vault/` — core truth entries
- `002-truth-topology/` — topology and map rules
- `003-machine-readable/` — machine-readable graph and schemas
- `004-governance/` — governance integrity standards
- `005-origins/` — origins, milestones, and external feedback
- `docs/` — live interactive map prototype
